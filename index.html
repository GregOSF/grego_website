THE GIVING TREE

Official Christmas 2015 donation page for BeSomeOne Worldwide, a non-profit helping providing education for the Maasai village of Terrat, Tanzania. Through fundraising and volunteer initiatives, the organization completed the construction of Namelok (which means “sweet school” in Swahili). Through the Namelok "Giving Tree" page, those looking to donate to the village can choose to give either to Maasai children's education, teachers and operations, or the Namelok Water Project.

  • Languages: Javascript, HTML5, CSS
  • Other: Integration with pre-existing non-profit SquareSpace application and Stripe API

AFTERSHOCK

Mobile (responsive), Ruby on Rails app designed to help users in the event of a major earthquake. Allows users to view and/or create their own custom pre- and post-quake checklists, utilizes geolocation technology to locate a user and the closest hospitals and triage centers, and sends an SMS text message to an emergency contact notifying of a user's status and safe meeting spot.

  • Languages: Ruby, Javascript, HTML5, CSS
  • Frameworks: Ruby on Rails, Bootstrap
  • External APIs: Twilio, Mapbox, Foursquare
  • Database: PostgreSQL
