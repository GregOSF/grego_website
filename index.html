AfterShock Mobile, Ruby on Rails app designed to help users through a major earthquake or other natural disaster. Allows users to view and/or create their own custom pre- and post-quake checklists, utilizes geolocation technology to locate a user and the closest hospitals and triage centers, and sends an SMS text message to an emergency contact notifying of a user's status and safe meeting spot. • Languages: Ruby, Javascript, HTML5, CSS • Framework: Ruby on Rails • External APIs: Twilio, Mapbox • Database: PostgreSQL
Nabe Nabe is a city-to-city neighborhood comparison app, allowing users to compare "nabes" in one city to others in cities with which they are familiar. Neighborhoods contain pre-seeded tags that are used to compare against each other. This project currently only compares a few select neighborhoods in San Francisco and New York.
